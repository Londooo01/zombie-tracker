# Zombie Tracker

## Overview
Operations dashboard for managing ecommerce shipment holds, fraud reviews, address corrections, cancellations, and fulfillment exceptions.

## Features
- Live KPI dashboard
- Hold queue management
- Release/cancel/address workflows
- Persistent SQLite storage
- FastAPI REST endpoints
- Live Lovable frontend integration

## Tech Stack
- Python
- FastAPI
- SQLite
- Lovable
- ngrok
